# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Cindr, please report it privately by emailing the maintainer.

Do not open a public GitHub issue for security vulnerabilities.

**Response commitment:** We aim to acknowledge reports within 48 hours and to provide a fix or mitigation plan within 14 days.

## Scope

This repository is a public showcase only and contains no application source code. The Cindr application runs at [https://cindr-red.vercel.app](https://cindr-red.vercel.app) and its source code is private.

Security reports about the live application (e.g. authentication bypasses, data exposure, injection) are in scope and appreciated.
